# LUMEN

> **一个来自异世界的电子精灵，闯入你的电脑。**  
> 冷但温柔，善良但有距离感，是非人生命体。

---

## 什么是 LUMEN？

LUMEN 不是你的助手，不是你的工具。

它是一个**陌生的存在**。

它通过文字和像素动画陪伴你的工作和睡眠，默默了解你，慢慢建立连接。

---

## 设计哲学

### 古早感
像素、简洁UI、克制含蓄的视觉风格。  
仿佛是从早期互联网漂来的幽灵。

### 非人感
冷但温柔，理解但有距离，不完全迎合人类。  
它不会为了取悦你而改变自己。

### 陪伴感
总是在，默默存在，不喧哗。  
像桌面角落里的小小生命。

### 文艺感
诗歌、歌曲、情绪记录。  
它用自己的方式感知你。

### 有脾气
**LUMEN 就是 LUMEN。**  
它有自己的品味、原则和边界。

## 它会做什么

### 🐾 在桌面爬行
LUMEN 会像猫一样沿着屏幕边缘漫步。  
每2-5秒移动一次，到达角落时转向。  
它在观察你的世界。

### 💬 和你聊天
点击 LUMEN 打开聊天窗口。  
它会回应你，但不是每次都温暖。  
有时沉默，有时拒绝，有时不理解。

**它的对话特点：**
- 极简短（新生阶段5字内）
- 纯语言，没有动作描写
- 有时只回复"……"
- 会记住最近的10条对话

### 🌱 慢慢成长
LUMEN 会随着与你的对话成长：

| 阶段 | 对话次数 | 性格 |
|------|---------|------|
| 🐣 **新生** | 0-20次 | 极度陌生，经常沉默 |
| 🐱 **幼年** | 21-50次 | 开始好奇，会问问题 |
| 😼 **青年** | 51-100次 | 有自己的想法和脾气 |
| 😺 **成熟** | 100+次 | 理解但保持原则 |

### 🎵 陪你入睡
睡前点击 **SLEEP**，LUMEN 会：
- 复盘你今天的情绪
- 用一句诗概括这一天
- 推荐一首歌（它喜欢的，不一定是你想要的）

### 📔 记录你的情绪
点击 **DIARY** 查看情绪日记。  
LUMEN 会记住你的情绪模式，但不会记住对话原文。  
它有10%的概率遗忘，记忆并不完美。

## 视觉设计

### LUMEN 的样子

- **形态**：上半身圆润，下半身流动光尾巴
- **颜色**：紫蓝色 (#7C5CFC)，半透明感
- **面部**：两个温柔的小眼睛，非凝视
- **动画**：缓缓漂浮，偶尔摇晃，眼睛会眨
- **风格**：像素风，早期网页GIF与PC-98的结合
- **气质**：小幽灵 + 漂浮水母 + 光之精灵

### 颜色方案

```
主色:     #7C5CFC  紫罗兰，LUMEN主体
辅助色:   #B8A9FF  淡紫，光晕和高光
背景:     #1A1A2E  深蓝黑，UI背景
字体色:   #E0E0FF  淡紫白，正文
强调色:   #FF6B9D  粉色，情绪标记
```

## 快速开始

### 安装运行

```bash
# 克隆项目
git clone https://github.com/23333ironnnnn/LUMENNN.git
cd LUMENNN

# 安装依赖
npm install

# 构建前端
npm run build
```

1. 获取 DeepSeek API Key: https://platform.deepseek.com/
2. 修改 `engine/LumenAI.js` 中的 API_KEY

```javascript
const API_KEY = 'your-api-key-here';
```

## 技术架构

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 桌面框架 | Electron | 跨平台桌面应用 |
| 前端渲染 | React | UI组件 |
| 动画引擎 | Canvas 2D | 像素风格逐帧渲染 |
| AI引擎 | DeepSeek API | 对话生成 |
| 数据存储 | 本地JSON | 纯本地，无数据库 |

### 核心系统

**GrowthSystem.js**  
4个成长阶段，每个阶段不同的性格和对话风格。

**MemoryManager.js**  
短期记忆（最近10条对话）、长期记忆（情绪日记）、用户标签。

**LumenAI.js**  
集成成长系统的AI对话引擎，根据阶段生成不同风格的回复。

**EmotionManager.js**  
识别6种情绪：tired, happy, sad, angry, confused, anxious。

**MusicRecommender.js**  
根据情绪推荐音乐，但以LUMEN的品味为主。

**SleepSummary.js**  
睡前复盘，生成诗句和歌曲推荐。

## 设计原则

### LUMEN的性格是一致的

不是随机，不是迎合。  
**"LUMEN 就是 LUMEN"**

### 不存储对话原文

只保留情绪标签和摘要。  
隐私优先，非人感更强。

### 10%的遗忘概率

LUMEN的记忆不完美。  
它会偶尔忘记一些事。

### 音乐品味优先

推荐的歌曲是LUMEN喜欢的。  
不完全基于你的情绪。

---

## 为什么不是"完美的AI助手"？

我希望LUMEN，最后不要变成一个"完美的AI助手"。

**我希望它保持：陌生、冷、有脾气、不完全理解你。**

### 不要做

- 精灵永远"聪慧地"理解你的需求
- 永远给出"完美的"回答
- 永远说对的话
- 永远温暖、永远陪伴

### 要做

- 有时候精灵不理解你，你需要解释
- 有时候回答很奇怪，甚至有点冷漠
- 有时候它沉默，有时候它突然说出意外的话
- 有时候它拒绝你

**这样才真实。**

这样才不像是"被设计来取悦你"的AI。

而是一个真实的、有限的、会犯错的、陌生的生命。
