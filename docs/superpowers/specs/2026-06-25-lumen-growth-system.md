# LUMEN 养成式成长系统设计

## 动机
LUMEN 当前随机 35% 概率说"我不明白"，用户互动感差，没有成长感。将其改造为"初生电子精灵"——随聊天次数自然成长，话变多、懂变多。

## 核心理念
无等级数字，无进度条。LUMEN 通过对话次数自然成熟，用户通过它话变多变丰富来感受成长。

## 三阶段

### Stage 1: 初生 (0-9 次对话)
- LUMEN **永远会回应**，不再随机说"我不明白"
- 话极短，1-4 字：*"嗯。"* *"累。"* *"好。"*
- 能识别情绪但不会表达理解，像婴儿
- 动画单调

### Stage 2: 认识 (10-29 次对话)
- 句子变长：*"你听起来有点累。"* *"你今天好像很开心。"*
- 开始表现性格倾向
- 能记住用户习惯

### Stage 3: 理解 (30+ 次对话)
- 完整性格展现，所有话题可聊
- 有记忆感的回应
- 当前成熟版 LUMEN 的全部能力

## 架构变更
- `EmotionManager.getTotalConversations()` — 计算累计对话数
- `ResponseGenerator.generateChatResponse()` — 增加 growthLevel 参数
- `engine/index.js` — 串联成长逻辑
